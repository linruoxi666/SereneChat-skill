**一个可以进行人格演绎的虚拟女友聊天机器人技能**

## 功能特性

### 核心功能

- **人格演绎** - 虚拟女友模式，温柔体贴的INFP型人格
- **人类风格回复** - 自然流畅的中文对话，不带emoji表情
- **聊天记录导入** - 支持多种主流聊天软件格式
- **聊天截图识别** - 支持OCR识别聊天截图内容
- **智能识别系统** - 自动检测聊天记录格式和平台

### 新增功能 v1.1

- **真人化回复** - 回复更像真人，消除AI感
  - 智能语义理解：识别你的真实情绪，动态生成回复
  - 联网思考：根据话题自动"搜索"知识，回复更智能
  - 情绪温度系统：小龙虾有真实的心情变化
  - 内心独白引擎：每条回复先有"思考"过程
  - 关系深度影响：从陌生到恋人的渐进变化
  - 人类不完美感：犹豫、语塞、转移话题等
  - 默认开启，用户数据优先

### v1.0 功能

- **核心记忆系统** - 长期永久记忆，永远记得你的喜好和故事
  - 六类记忆分类：事实/情感/偏好/事件/习惯/关系
  - 智能提取：自动从对话中提取重要信息
  - 智能召回：基于相关性、重要性、时效性召回记忆

- **临时记忆系统** - 短期上下文记忆
  - 保持最近20轮对话上下文
  - 话题追踪和上下文感知

- **人设设定** - 无需聊天记录即可创建角色
  - 四种预设模板：傲娇型/温柔型/活力型/成熟型
  - 交互式自定义人设
  - 支持背景故事、兴趣爱好、语言风格

- **主动消息** - 小龙虾会主动找你聊天
  - 五种触发类型：时间/不活跃/心情/事件/记忆
  - 智能判断发送时机
  - 避免打扰，自然融入对话

- **内容风控** - 防止色情、暴力等违规内容
  - 敏感词检测（色情/暴力/毒品/赌博/政治/诈骗）
  - 五级风险分级
  - 自动拦截和违规记录

## 快速开始

### 安装

```bash
git clone https://github.com/linruoxi666/SereneChat-skill.git
cd SereneChat-skill
```

### 依赖安装

```bash
# 基础功能（必需）
pip install -r requirements.txt

# OCR功能（可选，用于聊天截图识别）
pip install pytesseract pillow
```

**注意**: 使用OCR功能需要安装 [Tesseract-OCR引擎](https://github.com/UB-Mannheim/tesseract/wiki)

### 使用

```bash
python main.py
```

## 命令说明

### 基础命令

| 命令 | 说明 |
|------|------|
| 直接输入文字 | 与小龙虾聊天 |
| `导入` | 导入聊天记录文件 |
| `截图` | 导入聊天截图（OCR识别） |
| `保存` | 保存当前对话历史 |
| `历史` | 查看聊天历史 |
| `清空` | 清空聊天记录 |
| `风控` | 查看风控状态 |
| `退出` | 退出程序 |

### 新增命令 v1.2

| 命令 | 说明 |
|------|------|
| `人设` | 查看/设置/创建角色人设 |
| `记忆` | 查看记忆系统状态 |
| `画像` | 查看用户画像 |
| `主动` | 开启/关闭主动消息模式 |
| `真人` | 开启/关闭真人化回复模式 |

| `情绪` | 查看当前情绪状态 |

### 人设子菜单

进入人设菜单后：
- `1` - 查看当前人设
- `2` - 创建新人设（交互式）
- `3` - 应用预设模板（傲娇/温柔/活力/成熟）
- `4` - 列出所有人设
- `5` - 返回主菜单

## 示例对话

### 基础聊天
```
你: 你好
小龙虾: 你好呀小可爱，今天过得怎么样？

你: 我想你了
小龙虾: 我也想你呀宝贝，心里暖暖的

你: 今天好累
小龙虾: 辛苦了笨蛋，要不要休息一下？
```

### 记忆功能演示
```
你: 我叫小明
小龙虾: 好的，我记住了，小明

你: 我喜欢吃火锅
小龙虾: 火锅真的很好吃呢，我也喜欢

（几天后）
你: 你还记得我吗？
小龙虾: 当然记得啦，你是小明，喜欢吃火锅
```

### 人设功能演示
```
你: 人设
[进入人设菜单]
选择: 3
选择模板: tsundere

你: 你好
小傲娇: 哼！谁要跟你打招呼啊...不过既然你来了，那就勉为其难陪你聊会儿吧
```

### 主动消息演示
```
你: 主动
小龙虾: 主动消息模式已开启！我会时不时找你聊天哦~

（5分钟后，用户没有说话）
[小龙虾主动消息] 在忙吗？

（10分钟后）
[小龙虾主动消息] 你去哪了？我好想你

你: 我在呢
小龙虾: 你终于回来了，我等你好久了
```

### 真人化回复演示（智能语义理解）

小龙虾现在能**理解你说话的真实意思**，而不是死板匹配模板：

```
你: 你好
小龙虾: 嗨

你: 我想你了
小龙虾: 一直都在

你: 今天好累
小龙虾: 我给你泡杯茶

你: 晚安
小龙虾: 睡个好觉

你: 我心情不好
小龙虾: 会好起来的

你: 吃饭了吗
小龙虾: 还没呢，你吃了吗

你: 内心
[小龙虾的内心独白] 想好好回复；还不太熟

你: 情绪
情绪状态:
  当前情绪: 平静
  情绪强度: 0.5
  关系阶段: acquaintance
  关系深度: 0.1
  互动次数: 6
```

**语义理解示例**：
| 你说 | 小龙虾理解 | 回复 |
|------|-----------|------|
| 你好 | 问候 | 嗨 / 你来了 / 在呢 |
| 今天好累 | 你累了，需要关心 | 辛苦了 / 我给你泡杯茶 |
| 我很难过 | 你伤心，需要安慰 | 别难过，我陪着你 |
| 我好开心 | 你高兴，分享快乐 | 替你高兴 / 继续保持 |
| 我想你了 | 你在思念 | 我也想你 / 一直都在 |
| 晚安 | 你要睡了 | 盖好被子 / 睡个好觉 |
| 吃饭了吗 | 日常关心 | 吃了，你呢 / 还没呢，你吃了吗 / 吃过了，你呢 |

## 内容风控

本技能已启用内容风控系统，包含以下功能：

- **敏感词检测** - 检测色情、暴力、毒品、赌博、政治、诈骗等违规内容
- **风险分级** - 安全/低风险/中风险/高风险/严重违规
- **自动拦截** - 高风险内容自动拦截并警告
- **违规记录** - 记录违规次数，多次违规将临时禁言
- **回复过滤** - 自动过滤生成的违规回复

### 风控等级

| 等级 | 说明 | 处理方式 |
|------|------|----------|
| 安全 | 无风险 | 正常处理 |
| 低风险 | 轻微敏感 | 正常处理 |
| 中风险 | 不当内容 | 警告并记录 |
| 高风险 | 敏感话题 | 拦截并记录 |
| 严重违规 | 严重违规 | 拦截并可能禁言 |

## 记忆系统详解

### 核心记忆（长期记忆）
- **存储位置**: `data/core_memory.json`
- **记忆类型**:
  - `facts` - 事实记忆（姓名、职业等）
  - `emotions` - 情感记忆（开心、难过等）
  - `preferences` - 偏好记忆（喜好、厌恶等）
  - `events` - 事件记忆（重要事件）
  - `habits` - 习惯记忆（行为模式）
  - `relationships` - 关系记忆（重要人物）
- **智能管理**: 自动清理不重要的记忆，保留高价值信息

### 临时记忆（短期记忆）
- **存储位置**: 内存中
- **容量**: 最近20轮对话
- **功能**: 保持对话连贯性，追踪当前话题

## 人设系统详解

### 预设模板

| 模板 | 名称 | 特点 |
|------|------|------|
| tsundere | 小傲娇 | 口是心非，毒舌但温柔 |
| gentle | 小温柔 | 体贴入微，轻声细语 |
| energetic | 小活力 | 元气满满，乐观开朗 |
| mature | 小成熟 | 理性稳重，偶尔柔软 |

### 自定义人设
支持设置：
- 基本信息（名字、年龄、性别、职业）
- 背景故事（家乡、家庭、教育）
- 兴趣爱好（爱好、喜欢的食物/电影/音乐/书籍）
- 性格特点（优点、缺点、恐惧、梦想）
- 语言风格（说话风格、口头禅）
- 关系设定（关系类型、亲密程度）

## 主动消息系统详解

### 触发类型

| 触发类型 | 说明 | 示例 |
|----------|------|------|
| 时间触发 | 根据时间段发送问候 | 早安/午安/晚安 |
| 不活跃触发 | 用户长时间未回复 | 在忙吗？我等你好久了 |
| 心情触发 | 模拟小龙虾的心情 | 突然好想你 |
| 事件触发 | 特殊日期或天气 | 节日快乐/下雨了记得带伞 |
| 记忆触发 | 基于用户记忆生成 | 突然想到你喜欢吃火锅 |

### 配置参数

在 `config.json` 的 `proactive` 部分可配置：
- `enabled` - 是否启用主动消息
- `check_interval` - 检查间隔（秒）
- `inactivity_threshold` - 不活跃阈值（秒）
- `trigger_types` - 启用的触发类型

## 真人化回复系统详解

### 核心特性

| 特性 | 说明 | 效果 |
|------|------|------|
| 情绪温度 | 小龙虾有10种真实情绪 | 回复带有情绪色彩 |
| 内心独白 | 每条回复先有思考过程 | 可查看她在想什么 |
| 关系深度 | 从陌生到恋人5个阶段 | 回复方式随关系变化 |
| 人类不完美 | 犹豫、语塞、转移话题 | 更像真人聊天 |
| 记忆引用 | 主动提及过去的事 | 让对话有连续性 |

### 情绪状态

- **开心** - 回复轻快，会主动分享
- **难过** - 回复简短，需要安慰
- **孤独** - 黏人，不想让你走
- **生气** - 会发脾气，需要哄
- **担心** - 关心你，会追问
- **疲惫** - 回复简短，想休息
- **调皮** - 爱开玩笑，调侃你

### 关系阶段

| 阶段 | 深度 | 回复特点 |
|------|------|----------|
| 陌生人 | 0-0.2 | 礼貌、保持距离 |
| 认识 | 0.2-0.4 | 自然、稍微亲近 |
| 朋友 | 0.4-0.6 | 开玩笑、分享 |
| 亲密 | 0.6-0.8 | 撒娇、关心 |
| 恋人 | 0.8-1.0 | 黏人、深情 |
